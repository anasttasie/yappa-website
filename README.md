# Yappa website

Landing page for Yappa, separate from the app repo. Matches the app's look and feel. For Apple Developer Program review (organizations need a functional website).

## Before you deploy

1. **Privacy policy** — Update the privacy link in `index.html`. Replace `href="#"` with your actual URL (e.g. from the yappa app repo's GitHub Pages: `https://yourusername.github.io/yappa/docs/`).

2. **Instagram** — Replace `https://instagram.com/yappa` with your handle.

## Deploy

- **Vercel / Netlify** — Connect this repo, deploy. Publish directory: `/` (root).
- **GitHub Pages** — Push to GitHub, enable Pages from main branch.
- **Any static host** — Upload the files.

## When you launch

Replace `<div class="download-btn">coming soon</div>` with an `<a>` linking to your App Store URL.
